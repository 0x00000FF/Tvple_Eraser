Tvple Eraser는 자동으로 필터링 해 주지 못하는 영상에 대해 사용자가 직접 필터를 작성할 수 있도록 기능을 제공합니다.

만약 보고 싶지 않은 영상이 있다면 밑에 있는 필터에 추가해보세요.

필터는 ,(콤마)로 구분되며, "티비플", "tvple"등이 포함된 영상은 자동으로 필터링 해 드리니 입력하지 않아도 됩니다.

티비플에 올린 자작영상이나 따로 차단을 회피하고 싶다면, 여기에 유튜브 동영상 ID를 입력해주세요. 마찬가지로 ,(콤마)로 구분됩니다.

동영상 ID 예시 : https://www.youtube.com/watch?v=d-fCeGrDfaC

유튜브가 새로운 버전의 디자인을 적용함에 따라, 기존의 Tvple Eraser v2가 제대로 작동되지 않아 v3를 추가하였습니다. 기본값은 v3이지만, 구버전 레이아웃을 아직도 이용중이신 경우 v2를 선택하시면 됩니다.
